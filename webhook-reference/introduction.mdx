## Bem-vindo

A API de Webhooks permite a integração com eventos essenciais relacionados a **checkout, pagamentos, carrinho abandonado e rastreamento de pedidos**. Abaixo estão os eventos suportados:

### 📌 **Eventos Disponíveis**
- **Checkout**
  - 📢 `HOOK` - Notificação de checkout iniciado  

- **Carrinho Abandonado**
  - 📢 `HOOK` - Notificação de carrinho abandonado  

- **Pagamentos**
  - 📢 `HOOK` - Pagamento aprovado  
  - 📢 `HOOK` - Pagamento recusado  
  - 📢 `HOOK` - Pagamento pendente  
  - 📢 `HOOK` - Pagamento estornado  

- **Rastreamento**
  - 📢 `HOOK` - Status do rastreamento atualizado  


## 🔒 Autenticação

Nossos os Webhooks não necessitam de autenticação
